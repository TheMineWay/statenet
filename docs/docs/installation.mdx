# Installation

You can install the [package from the NuGet registry](https://www.nuget.org/packages/TheMineWay.StateNet) using one of the following commands:

<Tabs>
  <TabItem value="net-cli" label=".NET CLI" default>
    ```powershell
    dotnet add package TheMineWay.StateNet
    ```
  </TabItem>
  <TabItem value="pkg-manager" label="Package manager">
    ```powershell
    NuGet\Install-Package TheMineWay.StateNet
    ```
  </TabItem>
  <TabItem value="paket-cli" label="Paket CLI">
    ```powershell
    paket add TheMineWay.StateNet
    ```
  </TabItem>
</Tabs>

You can also go to the [Releases](https://github.com/TheMineWay/statenet/releases) and download compiled library files.
